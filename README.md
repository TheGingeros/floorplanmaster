# FloorPlanMaster

Blender plugin for creating floor plans and interior shells faster and easier.

This Blender plugin was originally created by Oskar Wladar as a bachelor's thesis at FIT CTU in Prague. It is currently under active development, with a full release planned for the near future.

Made by Oskar Wladar (Ginger3D)

## Milestones

### V0.1 - Alpha release
Plugin has basic functionality and represents the basic core functions. Important tasks can be done with this version.
