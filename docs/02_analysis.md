# 2. Analýza
Analytická kapitola buduje kompletní obraz problémové domény a uživatelských potřeb, ze kterého přímo vychází návrhová a implementační rozhodnutí. Začíná doménovou analýzou a informačním přehledem (2.1) — definuje problém a zasazuje ho do kontextu parametrického modelování a existujících řešení. Poté identifikuje cílové skupiny a persony (2.2), definuje vstupy a výstupy addonu (2.3), popisuje reprezentativní scénáře použití (2.4), odvozuje prioritizované funkční a nefunkční požadavky (2.5) a uzavírá technickou analýzou relevantních API a architekturálních možností Blenderu (2.6).
