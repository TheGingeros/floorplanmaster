# 2.3 Vstupy a výstupy
Tato sekce popisuje, co addon přijímá jako vstup a co vytváří jako výstup. Vstupy zahrnují 2D referenční podklady, numerické zadání rozměrů a volné skicování; výstupy jsou 3D hmotový model, exportovatelná optimalizovaná geometrie a přehledná prostorová data o místnostech.
